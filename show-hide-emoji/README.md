# Scrimba Challenge - Emoji Show and Hide

### The challenge

The challenge is to build emojis container and wire the emojis to show/hide individually.

Users should be able to:

- View the optimal layout depending on device's screen size
- Be able to show/hide emojis as a group and individually

Advanced:

- Create the design
- Make a game from the challenge

Demo here: https://scrimba-challenges.vercel.app/
